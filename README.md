# Дашборд аналитики League of Legends

## Описание проекта

Данный проект представляет собой систему анализа рейтинговых матчей League of Legends.
Он включает сбор, обработку и визуализацию игровых данных с использованием Python и интерактивного дашборда на Dash.

Дашборд позволяет исследовать статистику игроков, матчей и чемпионов в разрезе регионов и рейтинговых лиг.

---

## Возможности

* Загрузка данных с учётом rate limits API (20 запросов в секунду, 100 запросов за 120 секунд)
* Извлечение статистики:

### По матчам:

* длительность
* режим
* версия игры
* временная метка

### По игрокам:

* PUUID
* регион
* лига
* LP (очки лиги)
* количество побед/поражений
* винрейт
* количество матчей за месяц

### По чемпионам:

* championId
* количество матчей
* победы/поражения
* винрейт
* средние убийства/смерти/помощи
* Сохранение данных в CSV:

  * `matches_data.csv`
  * `players_data.csv`
  * `champions_data.csv`

---

## Визуализации

Дашборд включает:

* Топ-чемпионы по винрейту
* Распределение LP игроков
* Диаграмма длительности матчей
* Scatter plot: kills vs deaths по чемпионам

## Запуск проекта

### 1. Сбор и обработка данных

```bash id="run1"
jupyter notebook pipeline.ipynb
```

### 2. Запуск дашборда

```bash id="run2"
python dashboard.py
```

После запуска дашборд будет доступен по адресу:

```
http://127.0.0.1:8050/
```
